# Androgynous Angels — Social & Caption Templates

These templates define how Androgynous Angels communicates on social platforms.

They are designed to:
- Preserve elegance
- Support visual content
- Avoid over-explanation
- Scale without dilution

Primary storefront:
https://www.stepforddolls.com

---

## General Rule

The image is the lead.
The text is the atmosphere.

Captions should never compete with visuals.

---

## TEMPLATE 1 — Minimal Elegance

Use when the image speaks for itself.

Example structure:

[One calm sentence about material or feeling]

Example:
Soft satin. Secret Sensations.

---

## TEMPLATE 2 — Sensory Focus

Use when highlighting texture or fabric.

Structure:
- One sensory sentence
- One reinforcing line

Example:
Smooth satin designed to move with you.  
Luxury is felt, not announced.

---

## TEMPLATE 3 — Suggestive

Use for playful or intimate visuals.

Structure:
- One evocative sentence
- One grounding sentence

Example:
A softer way to be seen.  
Designed with intention.

---

## TEMPLATE 4 — Product-Centered

Use when driving traffic to a listing.

Structure:
- One mood-setting sentence
- One product anchor

Example:
Designed for softness and confidence.  
Available now at stepforddolls.com

---

## TEMPLATE 5 — Brand Statement

Use sparingly.

Structure:
- One declarative brand line

Example:
Elegance is a choice.
