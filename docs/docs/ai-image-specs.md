- Product showcases

## Core Rules (Non-Negotiable)

- Adults only (18+)
- Two models only per image
- Ultra-feminine presentation
- Satin-forward materials must be clearly visible
- Vintage Stepford / Vintage satin lingerie & panties
- Tasteful, fashion-editorial tone
- Submissive satin sissy crossdrsser tone

If an image violates any of the above, it is discarded.
