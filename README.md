# Satin-Dolls

**Documentation & coordination hub for the Androgynous Angels website**

Primary storefront: https://www.stepforddolls.com

---

## Overview

This repository serves as the central reference point for building, maintaining, and evolving the **Androgynous Angels** brand and Shopify website.

It is intentionally kept lightweight and public-facing, focusing on:
- Documentation
- Brand standards
- Visual and AI image specifications
- Coordination between collaborators and tools

This repository does **not** contain live Shopify theme code or sensitive credentials.

---

## What This Repository Is For

- Brand documentation and standards
- Visual identity rules and references
- AI image and content specifications
- Shopify coordination notes
- Long-term continuity and clarity

---

## What This Repository Is NOT For

- Live Shopify theme files
- API keys or secrets
- Customer data
- Paid assets

Sensitive or operational code belongs in private repositories only.
